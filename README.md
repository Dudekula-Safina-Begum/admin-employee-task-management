# Admin-Employee Task Management (With Backend API and Routing)

## Description
A web application with **React frontend** and a **Node.js backend** using Express. Admins can assign tasks to employees via a task management system, and employees can view task assigned. The backend uses a RESTful API with **Express** for handling requests and responses.

## Features
- Admin can create, assign tasks.
- Employees can view  assigned tasks.
- Backend API routes handle task CRUD operations.
- Authentication (optional) using JWT for secure login.

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB 
- Authentication: JWT (JSON Web Token)
- API Routes: Express Router


### Run

- Frontend: npm run dev
- Backend:  nodemon app.js
